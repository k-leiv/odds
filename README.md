# Odds

REST Web API technical assessment

## Runtime dependencies

- Java 17
- Docker

## How to run

```shell
# Start Postgres DB
docker-compose up -d

# Windows
./mvnw spring-boot:run

# Unix
mvnw spring-boot:run
```

## Documentation

- Api-docs: http://localhost:8080/api-docs
- Swagger UI: http://localhost:8080/swagger-ui/index.html
